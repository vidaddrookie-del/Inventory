# Inventory App

A simple web-based inventory management app built with HTML, CSS, and JavaScript. 
Track items, quantities, and stock levels right in your browser.

## Features
- Add, edit, and delete inventory items
- Search and filter items
- Local storage - data saves in your browser
- Export inventory to CSV
- Mobile responsive design

## How to Use
1. Clone or download this repo
2. Open `index.html` in your browser
3. Start adding items to your inventory

No installation or server needed - it runs 100% client-side.

## Tech Stack
- HTML5
- CSS3 / [Tailwind/Bootstrap if you used it]
- Vanilla JavaScript
- LocalStorage for data persistence
